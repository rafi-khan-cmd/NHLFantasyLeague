# Databricks Notebooks

This directory contains Databricks notebooks for supply chain simulation and analysis.

## Notebooks

1. **Monte Carlo Simulation** - Runs Monte Carlo simulations for supply chain scenarios
2. **Forecasting** - Demand and supply forecasting using time series analysis
3. **Data Preparation** - Prepares data from Snowflake for simulation

## Setup

1. Upload notebooks to your Databricks workspace
2. Configure cluster with required libraries:
   - PySpark
   - NumPy
   - Pandas
   - Snowflake connector
3. Set up Snowflake connection parameters
4. Configure job schedules if needed

## Usage

Notebooks can be run:
- Manually from Databricks UI
- As scheduled jobs
- Via Databricks API (from Spring Boot backend)
